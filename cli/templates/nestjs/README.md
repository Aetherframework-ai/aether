Aether workflow project initialized with NestJS template.

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run start:dev
```

## Project Structure

```
src/
  workflows/     # Workflow modules
  app.module.ts  # Root module
  main.ts        # Entry point
```

## API Endpoints

- `POST /workflow/execute` - Execute a workflow
